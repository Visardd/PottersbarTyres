# PB Tyres - Website

A modern, responsive website for Potters Bar Tyres built with React and Tailwind CSS.

## Features

- **Hero Slideshow** - Auto-rotating background images with manual navigation controls
- **Responsive Design** - Mobile-optimized with smooth scrolling animations
- **Service Showcase** - 7 tyre services displayed in an interactive grid
- **Google Reviews** - Display 5-star ratings and customer testimonials
- **Contact Section** - Map integration, hours, phone, and social media links
- **Navigation** - Fixed header with mobile menu overlay
- **Social Links** - Instagram and WhatsApp integration

## Services Offered

- New & Part Worn Tyres
- Precision Balancing
- Puncture Repairs
- Tracking (HUNTER)
- Alloy Wheels
- Spacers
- Wheel Refurbishment

## Getting Started

### Prerequisites
- Node.js (v14+)
- React 18+
- Tailwind CSS


## Customization

**About Section Image:**
Find the `AboutSection` component and update the `src` URL.

**Logo:**
Update the logo in the `Navigation` component.

### Update Social Links

- **Instagram:** Change `https://www.instagram.com/pb.tyres/` to your handle
- **WhatsApp:** Change `https://wa.me/447961564194` to your number

### Modify Content

- Company address, phone, and hours are in the `Contact` component
- Review quotes can be updated in the `Reviews` component
- Service descriptions are in the `Services` component
